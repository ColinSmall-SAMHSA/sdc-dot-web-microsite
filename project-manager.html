SDC Benefits Overview

A Secure Data Commons (SDC) is a controlled-access cloud-based data environment that allows users to conduct analyses and do development and testing of new tools and software products. The SDC stores sensitive transportation data made available by participating data providers, and grants access to approved researchers to these datasets. The SDC also provides access to open-source tools and allow researchers to collaborate and share code with other system users. It is not intended to be an alternative to any local jurisdiction’s traffic management center or local data repository.

Cost of Operating SDC

Several cost drivers can affect the estimation of costs when determining to utilize the SDC platform or create a customized solution. Due to the complexity of the activities and factors an SDC Cost Estimator has been created and can be found on the SDC Starter Kit page. The SDC Cost Estimator is a spreadsheet that provides a tool to estimate the cost of operating the SDC from three different perspectives.

  • From a platform standpoint
  • From a project standpoint
  • From a user standpoint

Project Managers must evaluate costs from all three standpoints to assess their need to use the SDC. Costs are provided in three categories:

  • Cloud Consumption
  • IT Services
  • Enablement

Starting a Project Within SDC

Onboarding steps vary upon each project’s needs. The SDC has a plethora of training and onboarding materials that can be accessed from the SDC Starter Kit. Typically, the SDC admin team will work with the project owner to set up both the data providers and the analysts to get the project up and running.

Creating a New SDC

There are several resources available to assist DOT Project Managers in standing up their own SDC like solution. The first step is to visit the SDC Starter Kit on the public confluence site and review the:

  • Terraform Templates
  • SDC Administrator Training Modules

The Terraform Templates link to the public repository for the SDC and come with a readme file on running these templates to quickly setup the templatized infrastructure the SDC resides on for your own needs.

The SDC Administrator Training modules are constantly being updated with training videos on operational aspects of maintaining a SDC like creating a user in active directory, navigating the AWS management console, onboarding users. These standard operating procedures can be used and tailored to meet a project’s specific needs.
